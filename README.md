# Price Tracker

A clean, professional real-time price tracking application for cryptocurrencies, precious metals, and currencies powered by Yahoo Finance APIs.

## Features

- **Real-Time Data**: Live market prices from Yahoo Finance
- **Multi-Asset Support**: Track crypto, metals, currencies, and Big Mac Index
- **Clean Interface**: Professional, minimal design
- **No API Keys Required**: Free Yahoo Finance integration
- **Local Storage**: All data stored locally in browser

## Supported Assets

### Cryptocurrencies
- Bitcoin (BTC), Ethereum (ETH), Binance Coin (BNB)
- Cardano (ADA), Solana (SOL), Ripple (XRP)
- Polkadot (DOT), Dogecoin (DOGE), Avalanche (AVAX), Polygon (MATIC)

### Precious Metals
- Gold (XAU), Silver (XAG), Platinum (XPT), Palladium (XPD)

### Major Currencies
- EUR, GBP, JPY, CAD, AUD, CHF, CNY, INR, AED, BHD

### Economic Indicators
- Big Mac Index: USA, UK, Japan, EU, Canada

## Getting Started

1. Open `index.html` in a modern browser
2. Complete the setup wizard
3. Select assets to track
4. View real-time prices
